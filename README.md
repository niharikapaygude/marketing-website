# marketing-website
This project is a responsive and user-friendly website created for **Jaldurg Agro Resort & Camping**, located near Tikona Fort, Maharashtra. 

The website showcases key details such as package pricing, room photos and videos, authentic Maharashtrian food, restaurant views, indoor/outdoor activities, and reasons to choose the resort. Each section is built as a separate HTML page with image galleries and smooth mobile-friendly design.

It is built using pure HTML and is hosted for free using GitHub Pages. This project was created with the goal of promoting the resort online and giving guests a clear and beautiful preview of their stay.
